# Router Rules

Централизованное управление пользовательскими списками Podkop.

## Назначение

Репозиторий является единственным источником правил маршрутизации для всех домашних роутеров.

## Структура

- rules/ — исходные категории доменов
- generated/ — автоматически собранные списки
- scripts/ — служебные скрипты
- docs/ — документация

## Обновление

Изменить rules/*.txt

↓

python3 scripts/build.py

↓

git push

↓

на роутере

/root/sync_router_rules.sh

или дождаться cron.
