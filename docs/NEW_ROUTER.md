# Настройка нового роутера

1. Установить OpenWrt.
2. Установить Podkop.
3. Создать профиль OpenAI.
4. Проверить интернет.
5. Скачать sync_router_rules.sh.
6. Выполнить sync_router_rules.sh.
7. Проверить:

nslookup chatgpt.com

nslookup vk.com

curl -I https://chatgpt.com

8. Настроить cron.
